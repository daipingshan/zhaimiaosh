1. 什么是“宅喵生活APP”？

答：【宅喵生活】是一款为喜爱网购的朋友们打造的，领券省钱，分享赚钱的APP；我们与百万厂商合作，每天推送近万件无门槛的大额优惠券，领到就是赚钱！优惠力度堪比双十一，一折爆款天天有！

2. 如何领券购物？

答：1.直接浏览商品，想要买的点击【自买可省】，淘宝的商品跳转淘宝点击【立即领券】/拼多多的商品跳转拼多多，点击【一键抢券】/京东的商品跳转京东点击【立即领取】，跳转商品详情页购买即可。

2.在app内搜索商品关键字，点击【搜索】，选择喜欢的商品点击【自买可省】，直接跳转领券购买，如果没有找到心仪的产品，可以切换搜索商品平台，搜索京东、拼多多等。

3.在淘宝天猫或京东、拼多多选好商品，一键复制‘’宝贝 标题‘’后，进入宅喵生活app进行搜索，剪贴板会自动展示商品标题，选择搜索平台，即可查出这款商品是否有券，如果您是推广员，无论商品有没有券，购买都会有奖励佣金，间接帮您省钱。

3. 领券购物商品质量是否有保障？

答：领券商品与原价商品都是同样的东西哦~领取优惠券相当于做了一次减价活动，领到券就有参与活动的权利。价格减少但商品是与原价购买的商品品质一致，不必担心。

4. 同一个商品可以多次购买吗？

答：因为优惠券本身相当于商家做了一场减价活动，所以限制每一个淘宝账号针对一个商品只能够享受一次优惠券。所以，如果想买多件同一商品，只能享受一次全额力度。或者，您也可以把商品分享给朋友，让她们来帮你购买，你代付就好啦~

5. 为什么领券下单后，“推广收益”内并无显示？

答：通常情况下，下单后5~10分钟内才会显示出“推广收益”。如果很久都没有显示，则要想想自己付款时是否使用的支付宝红包等第三方优惠红包？如果有使用则订单被其它第三方截流，系统自然追不到您的订单。 通常遇到这种情况，您可以及时取消删除订单，然后重新领券下单既可。（大家最好在付款时避免用支付宝红包等，直接用自己的卡内金额付款，避免不必要的麻烦。）

6. 如何通过宅喵生活APP挣钱？

答：1.进入【我的】，点击【分享app】，成功邀请一人注册，您就升级为推广员，无论您下单购买还是邀请的下级下单购买，您都可以获得推广佣金。邀请的人越多，等级越高，佣金越多。

2.点击【宅喵说】，进入列表，点击商品图片左下角的【*w人已分享】按钮，分享至朋友圈，有用户通过您分享的链接购买您就可以得到推广佣金。

3.进入商品详情，点击【分享可赚】，分享商品出去，只要有用户通过您分享的链接下单，您就可获得推广佣金。

7. 如何将宅喵生活APP分享给朋友？

答：点击“我的”——“分享app”，系统会随机给您一个邀请码，您可以修改邀请码，只能修改一次，修改完可以直接分享海报给好友，让好友扫海报中的二维码进行下载注册。也可以让好友直接在手机软件商城中搜索“宅喵生活”直接下载。注册时记得提醒对方输入你的邀请码哦~

8. 为什么“本月结算预估”中有金额，可是账户余额却没钱？

答：您只要下单，则佣金都被算入“本月结算预估”。只有在订单确认收货，并且过了10天后，这笔订单的佣金才会进入账户余额。账户余额大于20元时才能够提取哦！
